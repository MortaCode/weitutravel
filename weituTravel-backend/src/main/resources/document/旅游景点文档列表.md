# 旅游景点文档列表 (Markdown格式)

## 北京景点

### 故宫博物院
- **类别**: attraction  
- **城市**: 北京  
- **热度**: high  
- **详情**: 北京必游景点，明清两代皇家宫殿。开放时间：8:30-17:00（旺季），8:30-16:30（淡季）。门票价格：旺季60元，淡季40元。建议游玩时间：3-4小时。最佳参观路线：午门→太和殿→乾清宫→御花园→神武门。

### 长城（八达岭段）
- **类别**: attraction  
- **城市**: 北京  
- **类型**: historic  
- **详情**: 世界文化遗产，明代长城精华段落。开放时间：6:30-19:00（旺季），7:30-18:00（淡季）。门票价格：旺季40元，淡季35元。索道单程100元，往返140元。建议游玩时间：3-5小时。最佳季节：4-10月。

### 颐和园
- **类别**: attraction  
- **城市**: 北京  
- **严重程度**: garden  
- **详情**: 中国现存最大的皇家园林。开放时间：6:30-20:00（旺季），7:00-19:00（淡季）。门票价格：旺季30元，淡季20元。联票：旺季60元，淡季50元。推荐景点：万寿山、昆明湖、长廊、佛香阁。

## 上海景点

### 上海迪士尼乐园
- **类别**: attraction  
- **城市**: 上海  
- **类型**: themePark  
- **热度**: veryHigh  
- **详情**: 中国内地首个迪士尼主题公园。开放时间：8:30-21:30。门票价格：常规日435元，特别高峰日769元。推荐项目：创极速光轮、飞跃地平线、加勒比海盗。建议提前下载官方APP查看排队时间。

### 外滩
- **类别**: attraction  
- **城市**: 上海  
- **类型**: landmark  
- **价格**: free  
- **详情**: 上海的标志性景观，黄浦江畔。最佳游览时间：晚上19:00-22:00，夜景最美。可观赏陆家嘴金融区摩天大楼群。免费开放。建议游览路线：外滩→南京路步行街→人民广场。

### 东方明珠塔
- **类别**: attraction  
- **城市**: 上海  
- **类型**: observation  
- **详情**: 上海地标建筑，高468米。开放时间：9:00-21:00。门票价格：观光层199元，旋转餐厅自助餐328元起。最佳观景时间：傍晚，可以看日落+夜景。

## 杭州景点

### 西湖
- **类别**: attraction  
- **城市**: 杭州  
- **类型**: lake  
- **价格**: free  
- **详情**: 杭州最著名的景点，免费开放。推荐路线：断桥残雪→白堤→平湖秋月→苏堤→花港观鱼→雷峰塔。最佳季节：3-5月（春季赏花），9-11月（秋季赏桂）。可租自行车环湖游览约2-3小时。

### 灵隐寺
- **类别**: attraction  
- **城市**: 杭州  
- **类型**: temple  
- **文化价值**: high  
- **详情**: 杭州著名佛教寺庙，千年古刹。开放时间：7:00-18:00。门票价格：飞来峰景区45元，灵隐寺香花券30元。建议早上去，避开人流高峰。注意着装得体，保持安静。
